Detailed Description

The namespace corresponding to the Math module.

Math is the mathematics library for Simul. It contains Vector, Matrix and Quaternion classes, vector-matrix, vector-quat and matrix-quat functions, and some low-level maths functions.

Static library: (SIMUL)/lib/(PLATFORM)/(COMPILER)/(ConfigurationName)/SimulMath_(RUNTIME).lib

Dynamic library: (SIMUL)/exe/(PLATFORM)/(COMPILER)/(ConfigurationName)/SimulBase_(RUNTIME).dll

The Math library depends on the Simul/Base library.
