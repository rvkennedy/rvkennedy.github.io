Render trueSKY to the current render target.

This renders the trueSKY frame. It can be called (UE4-style) direct from the engine-side plugin wrapper, or (Unity-style) For UE4 we can call the whole render with all the information it needs. But for Unity we must first set the values, then wait for UnityRenderEvent() to be triggered.

Update a skylight cubemap and its corresponding spherical harmonics. Copy the specified skylight cubemap texture and spherical harmonic values from trueSKY to the external engine. This function has latency - it won't do anything the first time it is called, nor (on most platforms) the second or third.
