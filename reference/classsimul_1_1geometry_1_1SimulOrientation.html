Detailed Description

A spatial orientation comprising position and rotation.

The orientation holds position and rotation. Position is a vector which can be accessed using GetPosition(). Rotation is stored as a quaternion q. To transform use LocalToGlobalPosition, GlobalToLocalPosition, LocalToGlobalDirection and GlobalToLocalDirection.
