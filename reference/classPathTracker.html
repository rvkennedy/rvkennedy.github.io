Detailed Description

template<class ownerType, class nodeType, class dataType>
class PathTracker< ownerType, nodeType, dataType >

PathTracker: A template for a node that keeps a list of child nodes of a particular type.

PathTracker is a node that looks through its children for nodes of a certain type (nodeType), and keeps a map of them with corresponding instances of a helper type (dataType). dataType contains information particular to the Selector; while nodeType has no knowledge of Selectors that refer to it. When TimeStep is called, it iterates through the list calling HandleNode(nodeType*,dataType*). Unlike Selector, PathTracker keeps track of the node path between its owner and the selected node.
