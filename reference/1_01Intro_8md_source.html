1 1 Introduction {#man_1_introduction}
2 ============
3 
4 True Sky is the culmination of many years' research and development. It is based on physical principles
5 of light scattering and absorption, and is highly optimized for speed. We believe it is the best-performing system
6 for realistic realtime skies and clouds.
7 
8 The SDK comes as a set of libraries, under a main "Simul" directory. These can be linked statically or dynamically,
9 depending on the platform. Binaries reside in the directory "Simul/lib" in the case of static libraries,
10 and in "Simul/exe" for dll's. Source is found in the subdirectories for the constituent libraries.

54 The compiled Windows libraries are in two runtime versions, _MD and _MT.
55 The runtime is the set of libraries provided by Microsoft that provides basic system functions.
56 You should generally use the same runtime all the way through a project, although different runtimes can be used for DLL's,
57 because they are self-contained.
58 
59 For Windows, the Debug and Release configurations are supplied as DLL's linked against the dynamic, MD runtime.
60 The Static Debug and Static Release configuratons are supplied as static libraries linked against the static, MT runtime.
