Detailed Description

template<class resourceInterfaceType>
class simul::graph::meta::Resource< resourceInterfaceType >

A template for a class (that should also inherit simul::graph::meta::Group) that descendant nodes can use as a resource.

Resource is a node that looks through its children for nodes that are ResourceUsers of the correct type It stores a map of them. For as long as the resource is connected, resource users can call ResourceUser<resourceType>::GetResource() to obtain the resource's interface.
