Detailed Description

template<class T>
class simul::dx11::ConstantBuffer< T >

Useful Wrapper class to encapsulate constant buffer behaviour.

Usage: You must use the following member functions to use this buffer. On device restore, call RestoreDeviceObjects(m_pd3dDevice). After creating any effect that uses the buffer, call LinkToEffect(m_pSkyEffect,"BufferNameInFxFile"). On device shutdown, call InvalidateDeviceObjects(). Before applying the effect pass, call Apply(pContext)
