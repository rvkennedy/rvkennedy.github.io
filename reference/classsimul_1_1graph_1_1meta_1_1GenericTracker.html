Detailed Description

template<class ownerType>
class simul::graph::meta::GenericTracker< ownerType >

GenericTracker: A template for a node that keeps a list of child nodes of arbitrary types.

GenericTracker is a node that looks through its children for nodes of types that can be specified at a later time (nodeType), and keeps a map of them with corresponding instances of a helper type (dataType), which is also specified later. dataType contains information particular to the GenericTracker; while nodeType has no knowledge of Selectors that refer to it. When TimeStep is called, it iterates through the list calling HandleNode(nodeType*,dataType*).
