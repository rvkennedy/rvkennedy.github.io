#include <Simul/Platform/CrossPlatform/DeviceContext.h>

Detailed Description

The base class for Device contexts. The actual context pointer is only applicable in DirectX - in OpenGL, it will be null. The DeviceContext also carries a pointer to the platform-specific RenderPlatform. DeviceContext is context in the DirectX11 sense, encompassing a platform-specific deviceContext pointer
