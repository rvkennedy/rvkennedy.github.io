Detailed Description

An extension of SimulOrientation which describes a moving frame of reference.

The reference frame holds position, rotation, and linear and angular velocity. Velocity is a simul::math::Vector3 which can be accessed using GetVelocity(). Angular velocity is a simul::math::Vector3 which can be accessed using GetAngularVelocity().
