Detailed Description

template<class ownerType, class nodeType>
class SimpleTracker< ownerType, nodeType >

A template for a node that keeps a list of child nodes of a particular type.

SimpleTracker is a node that looks through its children for nodes of a certain type (nodeType), and keeps a map of them. Unlike Tracker, it does not use a user-defined helper type, instead passing the pointer to the node type.
