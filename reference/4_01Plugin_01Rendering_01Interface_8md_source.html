1 4 The Plugin Rendering Interface {#man_4_pri}
2 =========================
3 
4 The main way to connect trueSKY to a game or simulation engine is using the Plugin Rendering Interface.
5 A dynamic link library or dll called PluginRenderInterface.dll (or similar) is loaded, dynamically, from within the game engine.
6 The exported functions are used to perform sky rendering per-frame for each view.
