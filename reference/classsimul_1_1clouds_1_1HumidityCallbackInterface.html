Detailed Description

A callback class to control the distribution of humidity in a cloud node. The value returned by GetHumidityMultiplier is multiplied by the calculated cloud density. Derive your own class from HumidityCallbackInterface and implement GetHumidityMultiplier, then call AddHumidityCallback to attach it to the cloud node.
