Detailed Description

A fast cloud class.

Create an instance of FastCloudNode to generate cloud data for rendering. This class can use a custom memory allocator, because it is derived from a ResourceUser<> of simul::base::MemoryInterface. To use this feature call SetResource( simul::base::MemoryInterface *) with a pointer to a an instance of a class derived from MemoryInterface.
